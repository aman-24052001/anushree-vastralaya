## Features

| Feature | Details |
|---|---|
| 📒 Khata (Ledger) | Add credit sales per customer, track running balance |
| 💸 Partial Payments | Customers can pay in parts — ongoing cycle supported |
| 📞 One-tap Call | Green call button on every customer card → dials directly |
| 📷 Photos | Click photo of saree or save from gallery, auto-compressed |
| 📊 Dashboard | Total outstanding, monthly sales, 6-month bar chart, pie chart |
| 🔍 Search | Find customers by name or phone number instantly |
| 🇮🇳 Hindi / English | Toggle between languages with one tap |
| 📱 Offline | Works with zero internet after first load |
| 💾 Local Storage | All data stays on the phone — IndexedDB, no server |

## How to Use (for Maa)

1. Open the link in **Chrome** on Android
2. Tap the 3-dot menu → **"Add to Home Screen"**
3. It will appear as an app icon on your phone
4. All data is saved on your phone automatically

## Tech

- Pure HTML + CSS + Vanilla JS — zero dependencies
- IndexedDB for local storage (supports photos, large data)
- Single file — works offline as a PWA
- No backend, no login, no internet required
